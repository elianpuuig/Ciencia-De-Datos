## 📂 Proyectos Destacados

### 📈 1. Inteligencia Financiera: Análisis y Simulación de Acciones (ALUA.BA)
Un análisis integral de la serie de tiempo de las acciones de la empresa metalúrgica Aluar, conectando datos reales mediante la API de Yahoo Finance para evaluar riesgo, estacionalidad y rentabilidad.

**Características del análisis:**
*   **Auditoría de Datos (EDA):** Generación de perfiles de calidad de datos automatizados.
*   **Detector de Anomalías Estadísticas:** Gráficos híbridos por capas para aislar ruedas financieras con picos de volatilidad que superan la desviación estándar móvil.
*   **Simulador de Retorno (ROI) Interactivo:** Desarrollo de una interfaz con widgets (Slider) y callbacks en `JavaScript` inyectados en `Bokeh` para simular el interés compuesto de un portafolio de inversión en tiempo real.
*   **Estacionalidad Macroeconómica:** Implementación de mapas de calor (Heatmaps) para identificar patrones de rendimiento alcista o bajista mensual.
